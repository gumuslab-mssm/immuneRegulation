Upload cis-eQTL Data

For performance reasons, ImmuneRegulation makes use of cis-eQTL data files that are compressed and indexed using tabix.

1. Format your cis-eQTL results data accordingly with the following tab-separated columns (see example input):

Chromosome Gene SNP-ID/Name SNP-Position SNP-Position P-value . .

3 CCDC80 rs2638022 112265776 112265776 4.93996E-10 . .

5 ERAP2 rs2287988 96237326 96237326 4.5617E-8 . .

Column Definitions:

Chromosome: chromosome number for the SNP, Gene: HUGO gene symbol for the gene associated with cis-eQTL, SNP-ID/Name: rsid for the SNP, SNP-Position: SNP Position in 'Chromosome', P-value: p-value associated with the cis-eQTL

2. Sort the above data based on genomic location, and then generate your compressed file and index file using tabix (refer to Tabix manual for details.)

3. Place your data files (i.e. compressed file and index file) in a web-accessible server that supports Cross-Origin Resource Sharing (CORS), and copy the link to your compressed file below:

Upload trans-eQTL Data

1. Please make sure that the data file you provide is a .tsv (tab-sepaeated text) file with the following columns (make sure you use the same column headers listed below; and see example input):

PValue SNPName SNPChr SNPChrPos Gene ProbeChr ProbeCenterChrPos

5.16389E-39 rs2249563 4 39455378 RPL9 23 23925996

8.39797E-34 rs4103004 3 20042834 EFHB 3 18996472

Column Definitions:

PValue: p-value associated with the trans-eQTL, SNPName: rsid for the SNP, SNPChr: chromosome number for the SNP, SNPChrPos: SNP Position in SNPChr, Gene: HUGO gene symbol for the gene associated with trans-eQTL, ProbeChr: chromosome number for the probe in gene expression, ProbeCenterChrPos: center position -within ProbeChr- of the probe in gene expression

2. Place your data file in a web-accessible server that supports Cross-Origin Resource Sharing (CORS), and copy the link to your file below:
